# Deep Resource Manager

A RimWorld mod that displays all discovered deep resources in an organized table with advanced filtering and drill management.

## Features

- **Comprehensive Resource List**: View all discovered deep resources in a detailed table
- **Resource Filtering**: Filter by resource type with persistent settings that save across game sessions
- **Drill Management**: See active drills and their status (idle, working, no power)
- **Permission Control**: Manage drill permissions (allowed/forbidden) directly from the window
- **Resource Tracking**: Track mineable amounts per deposit and per drill radius
- **Quick Navigation**: Click to jump to deposits and drills on the map
- **Mod Compatibility**: Supports both standard deep resources and treasure mod resources

## Installation

### Steam Workshop (Recommended)
Subscribe to the mod on Steam Workshop and enable it in RimWorld's mod list.

### Manual Installation
1. Download the mod
2. Extract the mod folder to: `RimWorld/Mods/DeepResourceManager/`
3. Enable the mod in RimWorld's mod list

## Usage

1. Click the "Deep Resources" button in the bottom toolbar
2. Browse all discovered deep resources in the table
3. Use the "Filter Resources" button to show/hide specific resource types
4. Expand deposit rows to see associated drills and their status
5. Click on deposits or drills to jump to them on the map
6. Use checkboxes to allow/forbid drilling operations

## Compatibility

- **RimWorld Version**: 1.6
- **Dependencies**: None required
